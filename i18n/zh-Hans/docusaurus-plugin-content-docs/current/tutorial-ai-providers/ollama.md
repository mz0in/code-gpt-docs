# Ollama

官方网站 https://ollama.ai

这是一个运行、创建和共享大型语言模型（LLMs）的开源项目。
## 连接Ollama模型
- 从以下链接下载Ollama: [ollama.ai](https://ollama.ai/)
- 安装Ollama并使用命令 ```ollama pull codellama``` 使用模型codellama
- 如果要使用mistral或其他模型，您需要将codellama替换为所需的模型。例如：```ollama pull mistral```
