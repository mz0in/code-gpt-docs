# 单元测试

## 描述
CodeGPT单元测试功能使您能够快速生成代码的单元测试。通过选择特定的代码段，您可以提示CodeGPT创建相应的单元测试，提高代码的可靠性和质量。

## 如何使用：
- 选择代码：突出显示代码片段
- 右键单击所选代码，然后选择 **CodeGPT: Unit Testing**

## 示例
选择代码，右键单击，然后选择 `CodeGPT: Unit Testing`
