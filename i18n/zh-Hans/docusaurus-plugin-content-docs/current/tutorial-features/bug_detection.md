# Bug Detection

## 描述
***CodeGPT: Bug Detection*** 功能使您能够主动识别和解决代码库中的问题。通过选择特定的代码段，您可以促使CodeGPT进行全面的错误或潜在问题分析。

## 如何使用：
- 选择代码：突出显示代码片段
- 右键单击所选代码，然后选择 **CodeGPT: Bug Detection**

## 示例
选择代码，右键单击，然后选择 `CodeGPT: Bug Detection`
