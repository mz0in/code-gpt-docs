# 代码文档

## 描述
CodeGPT代码文档功能使您能够轻松生成详尽的代码文档。通过选择特定的代码段，您可以提示CodeGPT创建详细而丰富的文档。

## 如何使用：
- 选择代码：突出显示代码片段
- 右键单击所选代码，然后选择 **CodeGPT: Code Documentation**

为您的代码库赋予良好结构的文档。增强当前和未来的开发工作的协作和理解。

## 示例
选择代码，右键单击，然后选择 `CodeGPT: Code Documentation`
