# Code Refactoring

## 描述
CodeGPT代码重构功能使您能够轻松优化代码库。通过选择特定的代码段，您可以请求CodeGPT执行智能重构操作，提高代码的可读性和效率。

## 如何使用：
- 选择代码：突出显示代码片段
- 右键单击所选代码，然后选择 **CodeGPT: Code Refactoring**

体验CodeGPT智能重构的强大功能。轻松精准地提升代码质量。

## 示例
选择代码，右键单击，然后选择 `CodeGPT: Code Refactoring`
